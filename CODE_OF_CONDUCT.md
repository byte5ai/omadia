# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**, without
modifications.

The full text is published at:

- HTML: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>
- Markdown: <https://www.contributor-covenant.org/version/2/1/code_of_conduct/code_of_conduct.md>

The covenant defines the standards we hold ourselves to as contributors,
the maintainer responsibilities for upholding them, and the enforcement
process for incidents in any project space (issues, pull requests,
discussions, code reviews, public events held under the project name).

## Reporting

Instances of unacceptable behaviour can be reported privately to:

> `info@omadia.ai`

Reports are reviewed by the project maintainers and handled with
discretion. We follow the **enforcement guidelines** chapter of the
covenant (correction → warning → temporary ban → permanent ban) when
deciding on a response.

For security vulnerabilities, see [`SECURITY.md`](SECURITY.md) — that
process is separate from conduct enforcement.
